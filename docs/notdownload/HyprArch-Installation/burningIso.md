# Copying over the Iso to a USB-Drive

After the Iso is Downloaded we need to put the contents of the Iso File onto a USB Thumb Drive. For that you need:
1. 1x USB Drive with a minimum of 8GB of storage
2. Program to brun the Iso to the Drive

# Program to Burn the USB Drive

What programms can you use to burn the Iso Image to the USB drive:
1. [BalenaEtcher](https://etcher.balena.io/) Works with Windows, MacOS and Linux
2. [Rufus](https://rufus.ie/en/#download) Works only with Windows

There are ofcourse there are more that just two programs but those are the ones i know of.   

Now this is Important **EVERYTHING!!** on this USB Drive will be deleted during the formating process.
