# How do i get the HyprArch Iso

To Download the Newest Iso of HyprArch you must go the the Download page [here](https://git.stuple.net/Stuple-Networks/HyprArch/releases/latest) or download it directly from the git server [here](https://git.stuple.net/Stuple-Networks/HyprArch/releases/download/v0.0.1/HyprArch-2026.03.15-x86_64.iso)

# Checking the SHA256 Sum
Linux:

```bash
sha256sum HyprArch-2026.03.15-x86_64.iso
```
Windows PowerShell:

```Powershell
Get-FileHash -Path "HyprArch-2026.03.15-x86_64.iso" -Algorithm SHA256
```

Expected: 5f7330b99f48e865f2c0d2d6a907823b6ca47a1b1c1c275c74aa5ffabe3deacd
