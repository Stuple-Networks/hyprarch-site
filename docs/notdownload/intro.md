# HyprArch Intro

What is HyprArch and in what cases is it better than Arch.

## HyprArch Mindset

HyprArch (shortfrom HyA) is a Modified Arch Linux Iso, that comes preconfigured with Hyprland (thats why Hypr-(land) Arch).  

The HyprArch Iso can still be used for Installing Arch Linux but with the Changed os-release file.

### Why should / shouldn't chose HyprArch

Why Should you chose HyprArch? Because it is easy to install and configure, and you still say "I use Arch btw". But Why shouldn't you chose HyprArch? Well because you won't learn

about your own OS and you won't learn how do fix problems or any other kind of Bugs/Glitches.
