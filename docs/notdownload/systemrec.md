# System Requirement for HyprArch

| Minimum          | Recomended          |
| ---------------- | ------------------- |
| CPU: any X86-64  | CPU: modern X86-64  |
| GPU: I-GPU       | GPU: ~8GB vRam      |
| Memory: 2GB*     | Memory: 8GB         |
| Storage: 20GB*   | Storage: 50-70GB    |
| Internet: yes    | Internet: yes       |



*The Minimum Requirements were all done via testing HyprArch in a VM
